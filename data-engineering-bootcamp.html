Sample Capstone Project

Title: Daily Sales Pipeline

Objective: Build a complete ETL pipeline that ingests 50 CSV files from Azure Blob Storage, processes them using PySpark in Databricks, and delivers clean, aggregated data for business reporting.

Core Requirements:

  1. Bronze Layer: Raw CSV ingestion with schema validation.
  2. Silver Layer: Data cleaning, deduplication, and transformation.
  3. Gold Layer: Aggregated sales metrics (daily, by region, product).
  4. Orchestration: Schedule pipeline daily via Databricks Workflows.
  5. Monitoring: Set up alerts for pipeline failures.
  6. Documentation: HLD, LLD, and user guide.
